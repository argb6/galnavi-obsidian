# 外链安全设计

> [!info] 概述
>
> GalNavi 作为导航站，核心动作是引导用户跳转外部站点。**主站与详情页均采用倒计时确认 + noopener + noreferrer**。

## 风险

1. **反向 Tab 劫持**：`window.opener` 可被恶意页改写原标签
2. **Referer 泄露**：暴露来源为 GalNavi
3. **误点**：无确认直接跳转

## 多层防护

### 1. noopener
```javascript
window.open(targetUrl, '_blank', 'noopener,noreferrer');
```
或 HTML：`rel="noopener noreferrer"`。断开 `window.opener`。

### 2. noreferrer
跳转不发送 Referer，保护隐私。

### 3. 3 秒倒计时确认
```
点外链 → preventDefault → 倒计时层(3秒) → 可取消 → 到时打开
```
详见 [[外链跳转脚本(Redirect倒计时)]]。

### 4. CSP connect-src
客户端 fetch 仅限同源，限制数据外泄。详见 [[内容安全策略CSP]]。

## 主站 vs 详情页

| 维度 | 主站「链接直达」| 详情页外链 |
|---|---|---|
| 跳转 | 3 秒倒计时 | 3 秒倒计时 |
| 绑定 | document 事件委托 | 逐个 `addEventListener` |
| noopener / noreferrer | ✅ | ✅ |
| 取消 | ✅ | ✅ |

主站用事件委托，动态卡片也生效。内部路由、锚点、同源链接不拦截。
