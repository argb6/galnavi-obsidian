# 圣器殿堂

> [!info] 路由
>
> `GET /nav/group/` — 独立 SSR 页面，展示神器、魔器、仙器三类特殊资源。使用独立的 D1 数据库（与主站站点数据分离）。

## 页面定位

圣器殿堂是 GalNavi 的**独立功能页面**，与主站导航系统分离：
- 主题化设计，基于三界奇幻背景
- 展示三类特殊器物：神器、魔器、仙器
- 每件器物可关联官网、详情及额外链接
- 独立 D1，与主站站点表完全分离

## 页面入口

- 主站过滤器标签文案为 **「圣器」** → `/nav/group/`
- Sitemap 收录，priority 0.8
- SEO：`<title>` 圣器殿堂；description 含神器 / 魔器 / 仙器总览

## 三大分类

| 分类 | CSS / id | 图标 | 主题色变量 |
|---|---|---|---|
| **神器** | `divine` | ⚔️ | `--gold: #fbbf24` |
| **魔器** | `demonic` | 🔥 | `--red: #ef4444` |
| **仙器** | `immortal` | ✨ | `--emerald: #10b981` |

## 外观（现源码）

- 底色 `--bg: #080a12`，金/红径向光晕叠在深色渐变上
- 字体：Google Fonts **Noto Sans SC** + **Outfit**（与主站系统字体栈不同）
- 圆角 `--radius: 16px`
- `.glow` / `.particles` 为 **`display: none !important`**（光晕与粒子层当前关闭，仅保留背景渐变氛围）

## 数据结构（殿堂 D1）

### resources 表

| 字段 | 说明 |
|---|---|
| `id` | 主键 |
| `category` | 神器 / 魔器 / 仙器 |
| `name` | 器物名称 |
| `official_url` | 官网 |
| `details_url` | 详情 |
| `link1` ~ `link3` | 额外链接 |

### 查询

```sql
SELECT * FROM resources ORDER BY category, id
```

查询失败时返回错误态页面（不缓存）；成功则正常缓存。

## 前端交互

- 卡片列表展示（按分类）
- 分类切换 + 名称搜索
- URL 状态同步（`history.replaceState`）
- 首次访问传说弹窗（本地已读标记）
- 外链经安全校验后 **3 秒倒计时跳转**

## 与主站对照

| 维度 | `/nav/` | `/nav/group/` |
|---|---|---|
| 存储 | 主站 D1（站点表）| 殿堂 D1（`resources`）|
| 数据方式 | 客户端拉取 API | 服务端渲染后下发 |
| 外链 | 倒计时跳转 | 同左 |
