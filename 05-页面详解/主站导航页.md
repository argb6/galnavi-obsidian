# 主站导航页

> ！路由
> 
> `GET /nav/` — 1.28 MB，GalNavi 的**核心页面**，承载所有资源导航功能。全内联 HTML + 客户端 SPA（hash 路由）。

## 页面定位

主站是用户实际使用 GalNavi 的地方：
- 浏览收录的站点/模拟器/工具
- 搜索、按分类/标签过滤
- 查看站点详情、安全跳转外链
- 查看站长推荐与轮播通知

## 页面结构

```
┌──────────────────────────────────────────────────┐
│ [logo] 站点  工具  模拟器  会社  汉化组  [搜索框] │  ← 导航栏 + 抽屉触发
├──────────────────────────────────────────────────┤
│ view-home      首页：轮播图 + 站长推荐 + 最近更新 │
│ view-site      站点卡片网格                      │
│ view-tool      工具卡片网格                      │
│ view-simulator 模拟器卡片网格                    │  ← hash 路由切换
│ view-company   会社卡片网格                      │
│ view-hanhua    汉化组卡片网格                    │
│ view-tags      标签云                            │
├──────────────────────────────────────────────────┤
│ [redirectOverlay]  外链跳转倒计时层（隐藏）       │
└──────────────────────────────────────────────────┘
```

## 六大视图（hash 路由）

| data-nav | hash | 标签 | 内容 |
|---|---|---|---|
| home | （空）| 首页 | 轮播 + 推荐 + 最近更新 |
| site | `#site` | 站点 | websites 类卡片 |
| tool | `#tool` | 工具 | tools 类（暂无数据）|
| simulator | `#simulator` | 模拟器 | simulators 类卡片 |
| company | `#company` | 会社 | company 类 |
| hanhua | `#hanhua` | 汉化组 | hanhua 类 |
| — | `#tags` | 标签 | 标签云 |

切换由 `navigateTo(page)` 实现，详见 [主应用逻辑脚本（卡片与交互）](主应用逻辑脚本（卡片与交互）.md)。

## 卡片结构（buildCard）

每张卡片：
```
┌────────────────────────────────┐
│ [图标]  名称（搜索高亮）        │
│        一句话简介               │
│ [tag1] [tag2] [tag3] ...       │  ← 最多7个，可点击筛选
│ [介绍详情]  [链接直达]          │  ← 有url才可点
└────────────────────────────────┘
```
- "介绍详情" → `/nav/detail/?item_key=xxx`（见 [详情与外链跳转](详情与外链跳转.md)）
- "链接直达" → 外部站点，被全局拦截器改为倒计时跳转[外链跳转脚本(Redirect倒计时)](外链跳转脚本(Redirect倒计时).md)

## 首页（home）特色

首页是默认视图，比其他视图丰富：
1. **Hero 轮播图**：顶部全宽轮播，4 秒自动切换（[轮播图脚本（Hero Carousel）](轮播图脚本（HeroCarousel）.md)）
2. **站长推荐**：来自 KV `featured`，含三层 fallback + 自愈（[主应用逻辑脚本（卡片与交互）](主应用逻辑脚本（卡片与交互）.md) 的 5. renderHomePage() —— 首页（含推荐 + KV 自愈））
3. **最近更新**：最近添加/更新的站点

## 搜索

- 顶部搜索框（`#navSearch`）
- 实时过滤：input 事件 → `filterByKeyword`（匹配名称 + 标签）
- 搜索时卡片名称**高亮关键词**（`<span class="search-highlight">`）
- 点标签徽章自动填入搜索框

## 移动端

- 汉堡菜单（`setupDrawer` / `closeDrawer`）
- 导航链接收入抽屉
- 768px 断点适配

## 元信息

- `<title>`：`GALNAVI`
- CSP：严格白名单（[内容安全策略 CSP](内容安全策略CSP.md)）
- `<meta name="color-scheme" content="dark">`：声明暗色主题
- favicon：内联 base64 PNG（不依赖外部请求）
