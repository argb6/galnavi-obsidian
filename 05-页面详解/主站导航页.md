# 主站导航页

> [!info] 路由
>
> `GET /nav/` — GalNavi 核心页面，全内联 HTML + 客户端 SPA（hash 路由）。

## 页面定位

- 浏览收录的站点 / 模拟器 / 工具
- 搜索、按分类 / 标签过滤
- 查看站点详情、安全跳转外链
- 查看站长推荐与轮播通知

## 页面结构

```
┌──────────────────────────────────────────────────┐
│ [logo] 站点  工具  模拟器  会社  汉化组  [搜索框]    │
├──────────────────────────────────────────────────┤
│ view-home      首页：轮播 + 站长推荐 + 最近更新     │
│ view-site      站点卡片网格                        │
│ view-tool      工具卡片网格                        │
│ view-simulator 模拟器卡片网格                      │
│ view-company   会社卡片网格                        │
│ view-hanhua    汉化组卡片网格                      │
│ view-tags      标签云                             │
├──────────────────────────────────────────────────┤
│ [redirectOverlay]  外链跳转倒计时层（隐藏）         │
└──────────────────────────────────────────────────┘
```

## 六大视图（hash 路由）

| data-nav | hash | 内容 |
|---|---|---|
| home | （空）| 轮播 + 推荐 + 最近更新 |
| site | `#site` | websites |
| tool | `#tool` | tools |
| simulator | `#simulator` | simulators |
| company | `#company` | company |
| hanhua | `#hanhua` | hanhua |
| — | `#tags` | 标签云 |

详见 「主应用逻辑脚本（卡片与交互）」。

## 卡片结构

- 图标、名称（可搜索高亮）、简介、标签（可点筛选）
- 「介绍详情」→ `/nav/detail/?item_key=xxx`
- 「链接直达」→ 外链，经倒计时确认（「外链跳转脚本(Redirect倒计时)」）

## 首页特色

1. Hero 轮播（KV）
2. 站长推荐（KV；空则标签 fallback）
3. 最近更新

## 圣器殿堂入口

主站过滤器标签 **「圣器」** → `/nav/group/`（[[圣器殿堂]]）。

## 搜索

- 顶部搜索框实时过滤（名称 + 标签）
- 关键词高亮；点标签可填入搜索

## 移动端

汉堡抽屉菜单，约 768px 断点。

## 元信息

- 暗色主题；主色 `--primary: #4f7cff`，背景 `--bg: #141e33`；CSP 见 「内容安全策略CSP」
- 卡片：半透明底 + `backdrop-filter: blur(18px)`，圆角约 20px
- favicon 内联

## 脚本组成

| 作用 | 笔记 |
|---|---|
| 数据预加载 | 「数据预加载脚本（D1载入）」 |

| 轮播 | 「轮播图脚本（HeroCarousel）」 |

## 数据来源

- 卡片：`/nav/api/nav`（D1）
- 轮播：`/nav/api/hero`（KV）
- 推荐：`/nav/api/featured`（KV）

详见 「请求与渲染流程」。
