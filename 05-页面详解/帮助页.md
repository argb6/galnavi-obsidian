# 帮助页


> ！路由
> 
> `GET /nav/help/` — GalNavi 使用指南：浮动目录、滚动高亮、锚点复制。

## 页面定位

帮助页面向**新手用户**解释如何使用 GalNavi：
- 如何搜索
- 如何理解卡片信息
- 如何进入详情
- 如何安全跳转
- 如何贡献/反馈

## 页面结构

现版为**浮动目录栈**（非旧版左右分栏 sticky 侧栏）：

```
┌────────────────────────────────┐
│  使用指南正文                   │
│  ## tags / legend / github…    │
│                                │
│              ┌──────────────┐  │
│              │ 目录 (浮动)   │  │
│              │ 返回主站      │  │
│              └──────────────┘  │
└────────────────────────────────┘
```

- `#floatStack`：浮动栈（`#floatToggle` 开合目录 + `#floatNav` 锚点 + 返回主站）
- 正文区 `.card` / 浮动控件均 `backdrop-filter: none`（与关于页同系的散文文档风）

## 外观

- 底色与关于页同系：`--paper: #141e33`，径向蓝光晕
- 标题 `titleColorFlow` 蓝青渐变流动
- 系统中文字体栈（不拉 Google Fonts）

## 章节锚点（基于 HTML id）

| 锚点 | 章节 |
|---|---|
| `#tags` | 标签说明 |
| `#legend` | 魔法传说 |
| `#github` | GitHub 加速 |
| `#guide` | 迷茫指引 |

> 旧版 `#hamburger` / `#sidebar` / `#overlay` 已废弃；现交互见 [[帮助页侧栏与锚点脚本]]。

## 交互脚本

帮助页的业务脚本实现三大功能，详见 [[帮助页侧栏与锚点脚本]]：

1. **浮动目录开合**（`#floatToggle` / `#floatStack`）
2. **滚动高亮**（IntersectionObserver，`rootMargin: -8% 0px -72% 0px`）
3. **锚点复制 toast**（clipboard API + execCommand 降级）

## 移动端适配

- 浮动目录默认收起，点「目录」展开
- 点外部区域关闭；返回主站入口在浮动栈内

## SEO

- sitemap 收录（priority 0.8）
- 各章节有 id，利于深链接分享

## 与主站的关系

- 帮助页是独立 SSR 页面（非主站 SPA 的一部分）
- 独立 URL，可单独被收录/分享
- 主站可链接到帮助页（如"使用说明"入口）

## 设计亮点

| 特点 | 价值 |
|---|---|
| IntersectionObserver 滚动高亮 | 高效、不卡顿 |
| 精细 rootMargin | 激活时机符合阅读直觉 |
| 锚点复制 | 便于分享特定章节 |
| 移动端自适应 | 浮动目录栈（非抽屉侧栏） |
| 深链接支持 | `#guide` 直达章节 |
