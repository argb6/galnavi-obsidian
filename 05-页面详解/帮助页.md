# 帮助页

> [!info] 路由
> `GET /nav/help/` — 622 KB。GalNavi 的使用指南页面，含侧栏目录、滚动高亮、锚点复制。

## 页面定位

帮助页面向**新手用户**解释如何使用 GalNavi：
- 如何搜索
- 如何理解卡片信息
- 如何进入详情
- 如何安全跳转
- 如何贡献/反馈

## 页面结构

```
┌──────────┬────────────────────────────┐
│ 侧栏 TOC  │  使用指南正文                │

│ # guide  │  ...                       │
│ # search │  ## search 搜索             │

└──────────┴────────────────────────────┘
```

## 章节锚点（基于 HTML id）

| 锚点 | 章节（推测内容）|
|---|---|
| `#guide` | 入门指南 |
| `#search` | 搜索使用 |
| `#cards` | 卡片信息解读 |
| `#detail` | 详情页使用 |
| `#tags` | 标签含义 |
| `#github` | GitHub 贡献/反馈 |

> 另有 `#hamburger`/`#sidebar`/`#overlay`/`#toast` 是 UI 元素，非内容。

## 交互脚本

帮助页的业务脚本（行175）实现三大功能，详见 [帮助页侧栏与锚点脚本](../03-部署的JS/帮助页侧栏与锚点脚本.md)：

1. **侧栏开合**（移动端汉堡菜单）
2. **滚动高亮**（IntersectionObserver，rootMargin `-8% 0px -72% 0px`）
   - 滚动到某章节时，侧栏对应链接高亮
   - 多章节可见时取最靠上的
3. **锚点复制 toast**
   - 点章节旁的锚点图标 → 复制该章节直链
   - 如 `https://galnavi.top/nav/help/#guide`
   - 现代 clipboard API + execCommand 降级

## 移动端适配

- 侧栏默认隐藏，汉堡按钮触发
- `innerWidth <= 768` 时点链接后自动收起侧栏
- 遮罩点击关闭

## SEO

- sitemap 收录（priority 0.8）
- 各章节有 id，利于搜索引擎理解结构
- 锚点 URL 可被直接分享（深链接）

## 与主站的关系

- 帮助页是独立 SSR 页面（非主站 SPA 的一部分）
- 独立 URL，可单独被收录/分享
- 主站可链接到帮助页（如"使用说明"入口）

## 设计亮点

| 特点 | 价值 |
|---|---|
| IntersectionObserver 滚动高亮 | 高效、不卡顿 |
| 精细 rootMargin | 激活时机符合阅读直觉 |
| 锚点复制 | 便于分享特定章节 |
| 移动端自适应 | 抽屉式侧栏 |
| 深链接支持 | `#guide` 直达章节 |
