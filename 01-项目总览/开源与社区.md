# 开源与社区

## 开源协议

GalNavi 采用 **MIT License** 开源（Copyright © 2026 GALNAVI）。

> MIT 是最宽松的开源协议之一：允许商用、修改、分发、私有化使用，仅需保留版权声明。

- 协议原文：[github.com/argb6/gal-navigation/LICENSE](https://github.com/argb6/gal-navigation/LICENSE)
- README 版权声明：
  > "本仓库仅为导航性质，收录的所有工具、网站链接均指向第三方资源。各资源版权归原作者或平台所有。"

⚠️ 注意区分：**仓库代码/数据**是 MIT，但**收录站点本身的内容**各自有独立版权。

## GitHub 仓库

- 地址：[github.com/argb6/gal-navigation](https://github.com/argb6/gal-navigation)
- 仓库的特殊性：**只存放数据资源，不存放网站源代码**（详见 [[04-数据与资源/GitHub 仓库的角色]]）
- 仓库内容：
  - `README.md` — 项目说明
  - `LICENSE` — MIT 协议
  - `docs/data.json` — 站点数据（线上 D1 的滞后镜像，26 条 vs D1 的 29 条）
  - `docs/hero/` — 轮播图素材（被 KV 引用）
  - `docs/mascot/`、`docs/emoji/` — 吉祥物与表情素材
  - `assets/` — icon / logo / favicon
- 截至 2026-07-08 共 52 次提交

## 贡献方式

README 明确鼓励社区参与：

> "发现好用的工具或优质网站，欢迎提交 Issue 或提交到我的邮箱。"

1. **提交新站点**：通过 GitHub Issue 或邮件推荐优质 ACG 资源站/工具
2. **反馈问题**：站点失效、信息有误、体验建议
3. **代码贡献**：理论上可 PR，但仓库无源码，主要贡献点在数据与素材

## 开源生态定位

GalNavi 在 ACG 开源生态中属于**工具型/基建型**项目：
- 与收录的开源站点（如真红小站、Touchgal、Hikarinagi、鲲galgame 等均标注"开源"标签）形成互补
- 自身不生产内容，而是**为开源站点导流**
