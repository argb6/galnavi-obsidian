# GalNavi 的作用与价值

> [!abstract] 概述
> GalNavi 的"作用"指它作为产品提供的具体功能能力；"价值"指它为用户和社区带来的实际收益。两者分别对应"能做什么"与"带来了什么改变"。

## 一、作用（功能能力）

### 1. 资源聚合与发现
- 统一收录 ACG 资源站、模拟器、工具、会社、汉化组
- 通过 [[05-页面详解/主站导航页|主站的六大视图]]（站点/工具/模拟器/会社/汉化组/标签）分类呈现
- 详见 [[04-数据与资源/收录站点分类（模拟器与网站）]]

### 2. 结构化站点信息
每条记录包含（见 [[04-数据与资源/data.json 数据结构]]）：
- `title` 标题、`short_desc` 一句话简介
- `tags` 标签（资源站/网盘类型/是否需魔法/是否含AD等）
- `icon_path` 图标、`url` 主站链接
- 详情页可展开"介绍详情"与"官方矩阵导航"（帮助文档/GitHub/论坛/社群）

### 3. 多维检索
- **分类筛选**：按 simulators / websites / tools / company / hanhua 切换
- **关键词搜索**：`filterByKeyword` 同时匹配名称与标签（[[03-部署的JS/主应用逻辑脚本（卡片与交互）]]）
- **标签过滤**：点击标签徽章联动筛选
- **站长推荐**：KV 配置的精选条目（[[02-网站架构/API 端点清单|/nav/api/featured]]）

### 4. 安全外链跳转
- 外部链接不直接打开，而是弹出 3 秒倒计时确认层（[[03-部署的JS/外链跳转脚本（Redirect 倒计时）]]）
- 使用 `noopener,noreferrer` 防止 referer 泄露与 tab 劫持
- 详见 [[06-安全与SEO/外链安全设计]]

### 5. 永久发布页（入口稳定）
- 入口页 `/` 作为"永久发布页"，提示用户收藏（Ctrl+D）
- 即使主站域名变更，发布页提供最新入口
- 详见 [[05-页面详解/入口页（永久发布页）]]

### 6. 信息传达
- 轮播图（Hero Carousel）展示通知/公告（[[03-部署的JS/轮播图脚本（Hero Carousel）]]）
- 帮助页提供使用指南（[[05-页面详解/帮助页]]）

## 二、价值（实际收益）

### 对用户
| 价值点 | 说明 |
|---|---|
| 节省时间 | 一站直达，无需多站点来回切换与认知 |
| 降低门槛 | 标签明确标注"新手友好""需魔法""含AD"，新手避坑 |
| 安全可信 | 倒计时跳转 + 无广告，减少误点与风险 |
| 入口稳定 | 永久发布页解决域名漂移问题 |
| 横向对比 | 统一卡片视图便于对比站点属性 |

### 对收录站点
- 获得曝光，尤其对"用爱发电"的小众站点
- 官方矩阵整合减少用户迷路，提升站点留存

### 对 ACG 社区
- **降低信息壁垒**：把分散的社区入口集中
- **促进入门**：对 Galgame 新手友好的推荐与说明
- **开源示范**：MIT 协议 + 公开数据，可被复用与借鉴

## 三、作用边界（不做什么）

- 不托管任何资源文件（仅导航）
- 不对收录站点内容负责（版权归原作者/平台）
- 不做评分社区（但会链接到如 VNDB 等评分站）
- 不存储用户数据（无账号体系，仅 sessionStorage 存弹窗关闭状态）
