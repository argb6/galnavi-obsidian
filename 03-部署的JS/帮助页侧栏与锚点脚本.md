# 帮助页侧栏与锚点脚本

> [!info] 位置
>
> 帮助页 `/nav/help/` 页内业务脚本。现版为**浮动目录**（非旧版固定侧栏 + hamburger）。

## 职责

1. 浮动目录开合（`#floatToggle` / `#floatStack`）
2. `IntersectionObserver` 滚动时高亮当前章节链接
3. 点击锚点 `#` 复制章节直链并 toast

## DOM 结构（现源码）

```html
<div class="float-stack" id="floatStack">
  <button type="button" class="float-toggle" id="floatToggle"
          aria-expanded="false" aria-controls="floatNav">目录</button>
  <nav class="float-nav" id="floatNav" aria-label="本页目录">
    <a href="#tags">标签说明</a>
    <a href="#legend">魔法传说</a>
    <a href="#github">GitHub 加速</a>
    <a href="#guide">迷茫指引</a>
  </nav>
  <a href="https://galnavi.top/nav/" class="float-home">返回主站</a>
</div>
```

> 旧笔记中的 `#sidebar` / `#hamburger` / `openSidebar` 已过时。

## 开合逻辑

```javascript
function setOpen(open) {
  stack.classList.toggle('open', open);
  toggle.setAttribute('aria-expanded', open ? 'true' : 'false');
  toggle.textContent = open ? '收起' : '目录';
}
// 点击目录按钮切换；点击外部关闭
```

## 滚动高亮（仍为 IntersectionObserver）

```javascript
var observer = new IntersectionObserver(function(entries) {
  // 取当前相交且最靠上的 section，切换 .active
}, { rootMargin: '-8% 0px -72% 0px' });
```

`rootMargin` 与旧版一致，高亮逻辑未变；观察目标来自 `#floatNav` 内锚点。

## 锚点复制

点击章节旁 `#`（`.anchor-icon`）→ `navigator.clipboard` 或 `fallbackCopy` → `showToast`（约 1.8s）。
